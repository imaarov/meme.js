## Usage
1. Clone the repository:
   ```bash
   git clone https://github.com/imaarov/meme.js.git
   ```
2. Install dependencies:
   ```bash
   npm i
   ```
3. Compile the project:
    ```bash
   tsc --project . 
   ```
4. Or use the ready js compiled files:
    ```bash
   node dist/index.js 
   ```
